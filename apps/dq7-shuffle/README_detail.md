# DQ7 对对碰洗牌记录工具

> Dragon Quest VII 珍爱之物卡牌顺序记录工具

**GitHub**: https://github.com/Ztachi/strawberrybear-tools/tree/main/apps/dq7-shuffle

---

## 操作指南

### 定义阶段

依次点击空白格子，记录当前卡牌顺序：

| 点击顺序    | 自动分配     |
| ----------- | ------------ |
| 第 1、2 格  | 相同图案 "1" |
| 第 3、4 格  | 相同图案 "2" |
| ...         | ...          |
| 倒数第 2 格 | ⭐ 五角星    |
| 倒数第 1 格 | 🃏 小丑      |

全部格子填满后，自动进入交换阶段。

### 交换阶段

支持三种交换方式：

| 方式     | 操作                            |
| -------- | ------------------------------- |
| **点击** | 点击第一张选中 → 点击第二张交换 |
| **拖拽** | 直接拖拽卡片到目标位置          |
| **键盘** | 鼠标悬停后按空格键或回车键      |

### 其他操作

- **切换难度**：点击顶部 Tab
- **重置**：点击底部「重置」按钮

---

## 功能亮点

### 1. 三种交换方式自由切换

- **点击交换**：简单直接
- **拖拽交换**：快速丝滑，能够应对最高难度
- **键盘交换**：鼠标悬停 + 空格键/回车键

> 三种模式互不影响，随时切换

### 2. 14 种专属配色

每个数字图案都有独特颜色，一眼就能区分。

### 3. 流畅动画

卡片交换平滑过渡，拖拽时卡片放大旋转。

### 4. 四种难度网格

| 难度 | 网格  | 格子数 |
| ---- | ----- | ------ |
| 简单 | 4 × 3 | 12 格  |
| 普通 | 4 × 4 | 16 格  |
| 困难 | 5 × 4 | 20 格  |
| 超难 | 6 × 4 | 24 格  |
